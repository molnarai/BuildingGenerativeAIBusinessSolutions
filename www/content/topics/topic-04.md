This session covers three significant concepts and tools for development and deploment of software solutions (including AI Solutions).

At the end of this session, your system should be ready to start building your AI project.

Time is given in class to work though the tutorials, though I recommend to start installation ahead of time.

#### Prerequisites:
- Use a person computer or laptop. You must have administrative permission to the device!
- Setup VPN (@GSU visit https://technology.gsu.edu/technology-services/cybersecurity/virtual-private-network/)
- Install an Integrated Development Environment (IDE), such as [Visual Studio Code (VSCode)](https://code.visualstudio.com/), [JupyterLab](https://jupyterlab.readthedocs.io/en/latest/), [Spyder](https://www.spyder-ide.org/), or simiar tool.
